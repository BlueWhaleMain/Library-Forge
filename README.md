# [Library-Forge](https://github.com/BlueWhaleMain/Library-Forge)

Minecraft Forge Mod

拓展与修改原版内容，以此获得更多玩法

任何对原版的修改，都是为了借此添加更多内容或游戏步骤

## 核心内容

模组的主题：Minecraft

模组是围绕Minecraft展开的

## 状态

不排除伴随原版更新对重复内容的删减，这是最好的结果
